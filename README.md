# Personal Portfolio Website

This repository contains the source code for my personal portfolio website. The site presents my background, technical skills, selected projects, and contact information.

## Project Overview

The portfolio is designed to provide a clear overview of my work and development experience. It includes sections for About, Skills, Projects, and Contact, along with links to my coding and professional profiles.

## Technologies Used

* HTML
* CSS
* JavaScript
* Font Awesome

## Features

* Structured layout with navigation sections
* Tab-based content for skills, projects, and education
* Links to GitHub, LinkedIn, and LeetCode profiles
* Contact form connected to Google Sheets for message storage
